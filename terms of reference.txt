Основные функции для MVP (Минимально жизнеспособного продукта):
Регистрация и авторизация пользователей

Создание учетной записи
Вход в систему (с JWT или без него)
Восстановление пароля (если нужно)
Каталог товаров

Список товаров с изображениями, названиями и ценами
Фильтрация (по бренду, категории, цене и т. д.)
Поиск товаров по названию
Страница товара

Фото, описание, цена
Кнопка «Добавить в избранное» (если нужно)
Корзина (если нужна)

Добавление товаров в корзину
Удаление из корзины
Оформление заказа (или просто сохранение товаров в профиле)
Панель администратора (если требуется)

Управление товарами (CRUD: создание, редактирование, удаление)
Управление пользователями

Юзер сервис (15 функций):
registration_db - регистрация
login_db - логин
change_account_db - изменение профиля
delete_account_db - удаление аккаунта
add_to_favorite  -добавление в избранное
add_to_cart - добавление в корзину
del_from_cart - удаление из корзины
del_from_fav - удаление из избранных
get_exact_prod_by_brand - список продуктов по бренду
get_exact_prod_by_cat - список продуктов по категории
get_prod_main_menu - главное меню (куча продуктов)
get_all_brands - список всех брендов
get_all_cat - список всех категорий
get_exact_category - посмотреть инфу об обной категории
get_exact_brand - посмотреть инфу об одном бренде
offer - Оформление заказа, заказ отсылается в группу в тг

Админ сервис (10 функций)
get_exact_user - посмотреть выбранного юзера
new_brand - добавление нового бренда
new_category - добавление новой категории
new_product - добавление нового продукта
change_prod_items - изменение инфы о продукте
del_exact_prod - удаление определенного продукта
change_br_items - изменение инфы о бренде
del_ex_br - удаление определенного бренда
change_cat_items - изменение инфы о категории
del_ex_cat - удаление определенной категории
